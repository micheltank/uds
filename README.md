Demo application to create tasks

# Backend

## Technology Stack:
- Java
- Spring
- MySql
- Lombok

## How run:
- Create and run database: docker-compose -f backend\mysql.yaml up
- Run the scripts inside backend\src\main\resources\modelagem.sql
- Run app: mvnw

# Frontend

## Technology Stack:
- Angular 6
- Material

## How run:
- Run: npm install
- Run app: npm start
